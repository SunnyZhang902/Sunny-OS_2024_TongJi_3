
# 🤖文件管理

实现了文件管理系统的模拟，实现了真存储

同济大学 软件工程 2024年春 操作系统大作业
## 安装

使用 npm 安装 my-project

```bash
1 将项目Git到本地
2 通过 conda 部署 Python 环境，Python 版本为 3.8
3 通过 pip安装 PyQt5
```
    
## 在本地运行

Clone 这个 project

```bash
  git clone https://github.com/SunnyZhang902/Sunny-OS_2024_TongJi_3.git
```

前往项目目录

```bash
  cd Sunny-OS_2024_TongJi_3
```

安装依赖

```bash
  pip install PyQt5
```
运行程序

```bash
  pip python main.py
```
